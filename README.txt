Lexical and Syntactic analyser

This program scann miniC# code, and verify if the tokens are valid 
in a miniC# context.

When you run the program a file dialog will display, so you 
can choose the file (.frag) after this 
a new dialog will show, asking if you want to analyze another 
file, if you do click YES and the file dialog will show again. 
In case you are done only click NO and the program will close.

If the file doesn't have errors it will show "Correct file" 
otherwise you will see the analysis report, showing the lexical
sintactic and lenght errors.
